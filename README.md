# render
前端快速渲染引擎
* ## render object
```html
<div class="root">
    {{item.name}}
</div>
```
```javascript
let data ={
    name:"tong",
    age:22
};
new Render(".root").obj(data);
```
---
* ## render array
```html
<ul class="root">
    <li id="list-{{key}}">{{item.name}} {{item.age}}</li>
</ul>
```
```javascript
let data =[
    {
        name:"tong",
        age:22
    }
]
new Render(".root").for(data);
```
### you can use `{{key}}` get `index` when use `for` function
